API Reference

The API was built with Express and MongoDB. It is a JSON API, and therefore all responses are JSON.


This documentation explains how to use the music find API.

Authorization

All API requests requires authorization. In order to get authorized a user has to be created, which can be done at ---. In order to get authorized a post request has to be made:

POST /api/login&username=User&password=Password
Parameter Type Description
username string Required
password string Required

Data

Artists

Getting all the artists in the database:

GET /api/get_all_artists

Getting all artists that contains the given parameter:

GET /api/get_artists/searchstring
Parameter Type Description
Search string string Not required. Returns all artists if empty

Albums

Getting all the albums in the database:

GET /api/get_all_albums

Getting all albums that contains the given parameter:

GET /api/get_albums/searchstring
Parameter Type Description
Search string string Not required. Returns all albums if empty

Songs

Getting all the songs in the database:

GET /api/get_all_songs

Getting all songs that contains the given parameter:

GET /api/get_songs/searchstring
Parameter Type Description
Search string string Not required. Returns all songs if empty
