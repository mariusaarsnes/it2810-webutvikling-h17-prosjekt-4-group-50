API Reference

The API was built with Express and MongoDB. It is a JSON API, and therefore all responses are JSON.


This documentation explains how to use the music find API.

Authorization

All API requests requires authorization. In order to get authorized a user has to be created, which can be done at ---. In order to get authorized a post request has to be made:

POST /api/login&username=User&password=Password
Parameter Type Description
username string Required
password string Required

Data

Artists

Getting all artists with names that contain the given parameter:

GET /api/artists/search_string/offset/amount

Getting all albums that contains the given parameter and sort ascending:

GET /api/albums_asc/search_string/offset/amount

Getting all artists that contains the given parameter and sort descending:

GET /api/artists_desc/search_string/offset/amount
Parameter Type Description
Search string String Not required. Returns all artists if no parameter
Offset Integer The first index of the result query to be returned.
Amount Integer The amount of objects to be returned.
